Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans : The main difference between those properties is realized through what they return. The innerHTML returns all the html text contents including html tags and spacing. The innerText returns only the text content of the element and its children without css hitten text spacing and tags. Finally the textContent property returns all the text contents of the element and its decendaces with spacing and css hidden text, but without tags.

Question-3: What is event delegation in the DDOM?

Ans : Event delegation is basically a pattern to handle events efficiently. With this process we can add a commont event listener condition the parent element and target its specific child or element instead of adding same event condition to every similar elements.

Question-4: What is event bubbling in the DOM?

Ans : Event Bubbling in javaScript is a mechanism where an event triggered on a child propagates upwards through its ancestors all the way to the parent elements. It allows the parent elements to respond to events triggered by their child elements.
